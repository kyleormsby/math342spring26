Theorem 2.6.3. Quotients that separate points.
Suppose \(q\colon X\to Z\) is a quotient map and define
\begin{equation*} E_q := \{(x,y)\in X\times X\mid q(x)=q(y)\}. \end{equation*}
Aside: Additional hypothesis!
Note the additional hypothesis — that \(X\) is compact — in the converse theorem. Find an example demonstrating that this additional hypothesis is necessary.
🔗
If \(Z\) separates points, then \(E_q\) is a closed subset of \(X\times X\text{.}\) If \(X\) is compact and \(E_q\subseteq X\times X\) is closed, then \(Z\) separates points.
